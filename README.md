# School_Projects

## Executive Summary
Using R to build statistical models that identify key drivers for ex: Car Price or Customer Churn.

## Tech Stack
- **Language:** R
- **Library:** `ggplot2`, `dplyr`, `stats`
- **Techniques:** Multiple Linear Regression, Data Cleaning (IQR Method), EDA

## Process
1. **EDA:** Visualizing distributions and correlations.
2. **Cleaning:** Removing outliers using the IQR method and handling missing values.
3. **Modeling:** Evaluating multiple models to find the best fit.
